Detailed Description

Class implements a PID Control Loop.

Creates a separate thread which reads the given PIDSource and takes care of the integral calculations, as well as writing the given PIDOutput

void PIDController::CallCalculate ( void *  controller)
staticprivate

Call the Calculate method as a non-static method. This avoids having to prepend all local variables in that method with the class pointer. This way the "this" pointer will be set up and class variables can be called more easily. This method is static and called by the Notifier class.
